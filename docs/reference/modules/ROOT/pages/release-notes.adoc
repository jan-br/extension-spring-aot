== Release 5.1.0-RC2

Major upgrade of the Spring AOT Extension to support Axon Framework 5, Spring Boot 4, and Spring Framework 7.

=== Highlights

- Upgraded to Axon Framework 5.1.0-RC2, Spring Boot 4.0.5, Spring Framework 7.0.6, Java 21
- Updated all package references for Axon 5's restructured module layout (`org.axonframework.messaging.core`, `org.axonframework.modelling.entity`, `org.axonframework.extension.spring`)
- Added dynamic scanning of Axon `*Definition` classes for reflection registration
- Added JPA entity hints for `TokenEntry`, `AggregateEventEntry`
- Added gRPC protobuf class scanning for Axon Server connector
- Replaced `SimpleEntityManagerProvider` with `FactoryBasedEntityManagerProvider` for native image compatibility
- Removed stale gRPC/Netty native-image configuration (gRPC 1.79+ ships its own)
- Removed `ResourceInjectorAutoConfiguration` (saga support removed in Axon 5)
- Updated Jackson dependency to Jackson 3 (`tools.jackson.core`)
- Updated JUnit to 6.0.3 for Spring Boot 4 compatibility

=== Breaking changes

- Requires Java 21 or later (was Java 17)
- Requires Spring Boot 4 (was Spring Boot 3)
- Requires Axon Framework 5.1 (was Axon Framework 4.x)
